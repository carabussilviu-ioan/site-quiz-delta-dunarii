Site Delta Dunării - Carabus Silviu-Ioan

Acesta este un proiect web educațional care prezintă frumusețea Deltei Dunării și oferă utilizatorilor o modalitate distractivă de a-și testa cunoștințele printr-un quiz interactiv.

🚀 Funcționalități
 Detalii despre atracții  + O secțiune dedicată peisajelor + Un test de cunoștințe implementat cu JavaScript

🛠️ Limbaje folosite:
HTML5 + CSS3 + JavaScript
Visual Studio Code Insider - IDE
